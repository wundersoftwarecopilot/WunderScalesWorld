# 🌍 WunderScalesWorld

A lightweight Progressive Web App (PWA) built with Vanilla JavaScript that aggregates multiple RSS news sources into a clean mobile-first interface.

## ✨ Features
- Multi-source RSS aggregation
- Category filtering (World, Tech)
- Mobile-first UI
- Offline support (Service Worker)
- Installable PWA

## 🚀 Run locally

```bash
npx serve .
```

or

```bash
python -m http.server
```

## 🌐 Deploy (GitHub Pages)
1. Go to Settings → Pages
2. Select branch: `main`
3. Save

Your app will be live at:
https://wundersoftwarecopilot.github.io/WunderScalesWorld/

## ⚠️ Notes
- Uses rss2json proxy (for CORS)
- For production, consider your own RSS proxy
